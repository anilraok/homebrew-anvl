# homebrew-anvl

[Homebrew](https://brew.sh/) tap for [Anvl](https://anvl.dev/) products.

## Install

```sh
brew install --cask anilraok/anvl/dwani
```

Or, if you prefer the two-step form:

```sh
brew tap anilraok/anvl
brew install --cask dwani
```

## Available casks

| Cask | Product | Description |
|---|---|---|
| [`dwani`](Casks/dwani.rb) | [Dwani](https://dwani.app/) | Push-to-talk voice dictation utility for macOS |

## Updates

```sh
brew update && brew upgrade --cask dwani
```
